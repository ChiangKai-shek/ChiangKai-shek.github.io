IDE内vim插件配置

1. 光标移动

  • command mode:
基本移动操作
h/j/k/l 左下上右
Ctrl + f/b 上页/上页
Ctrl + u/d 上半页/下半页
光标移动操作
zz 光标屏幕居中
zb 光标屏幕底端
zt 光标屏幕顶端
H 移动到当前屏幕第一行行首
M 移动到当前屏幕中央行行首
L 移动到当前屏幕最后行行首
$ 移动到行尾
0 移动到行首
^ 移动到行首第一个字符
代码折叠
zo 打开折叠
zc 关闭折叠
zR 打开全部折叠
zM 关闭全部折叠
  • insert mode:
移动
Ctrl + h/j/k/l 左下上右

2. 文本操作

  • command mode:
:n1,n2/word1/word2/g n1行到n2行替换word1为word2
:1,$s/word1/word2/g 或 :%s/word1/word2/g 第一行到最后一行替换word1为word2
:1,$s/word1/word2/gc 或 :%s/word1/word2/gc 第一行到最后一行替换word1为word2,并且每次替换前要求确认
J 当前行与下一行合并为1行
u 撤销操作
Ctrl+r 重做操作
. 重复前一个动作
  1. 窗口操作
分屏
sp 上下分屏
vs 左右分屏
Ctrl W + h/j/k/l 切换分屏左下上右
:only
或Ctrl W + o		 只保留当前分屏
Ctrl + ^ 最近两个文件间切换
fin[d]/cla[ss] 打开IDEA查找功能files/class
Alt + ←/→ 多文件之间切换
